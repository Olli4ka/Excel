# Excel
Робота з Excel для обробки та підготовки даних

---
## 1 Автоматизація збору даних

У вас є текстовий файл data.txt, що містить рядки з числовими значеннями
(наприклад, "120\n 85\n 90\n ..."). Напишіть Python-скрипт, який зчитує дані 
з цього файлу, обчислює їхню суму та середнє значення, а потім зберігає 
результати у новий файл report.txt у форматі: "Сума: X, Середнє: Y"

---
## 2 Запис основного DataFrame в Excel

Напиши програму, яка створює DataFrame з даними про студентів: ім'я студента
та його середній бал. Запиши DataFrame в Excel-файл з ім'ям "students.xlsx".
Використовуй лист з ім'ям "Група 1". Програма повинна створити файл Excel з 
вказаними даними без збереження індексів.
